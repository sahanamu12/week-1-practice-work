do while loop:


1) 1) WAP to print 1-10 numbers using do while loop syntax.

begin
numeric num=1
do
{
print(num)
num++
}
while (num<=10);
end

2)  WAP to print 10-1 numbers

begin
numeric num=10
do
{
print(num)
num--
}
while(num<=10)
end
